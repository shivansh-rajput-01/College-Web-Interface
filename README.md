# College Web Interface with Backend Integration

A professional web portal for educational institutions, developed with a focus on seamless integration between a pre-designed frontend and a functional backend. This project showcases the process of adding server-side capabilities to a static web interface.

## Project URL
https://college-web-interface-5.onrender.com

## Key Features
- **Frontend-Backend Integration:** Successfully connected a detailed HTML/CSS interface with an Express.js backend.
- **User Authentication:** Functional Login and Signup modules to handle user entry and data submission.
- **Structured Academic UI:** Organized sections for departments, campus highlights, and institutional information.
- **Form Data Processing:** Logic to capture and process user inputs from the frontend to the server.
- **Responsive Layout:** Clean design built with Bootstrap for better accessibility across devices.

## Technologies Used
- **Backend Framework:** Express.js
- **Frontend:** HTML5, CSS3, JavaScript
- **UI Framework:** Bootstrap
- **Hosting:** Render

## Project Evolution & Security
This project started as a dedicated frontend interface and was later upgraded with Express.js for backend functionality. Currently, passwords are processed in plain text for demonstration; implementing secure hashing (Bcrypt) is the next logical step in development.

## Installation and Local Setup
To run this project locally:

1. Clone the repository:
   `git clone https://github.com/shivansh-rajput-01/college-web-interface.git`

2. Install dependencies:
   `npm install`

3. Start the server:
   `node index.js` (or `npm start`)

4. Access in browser:
   Open http://localhost:3000
